AngularJS: Ejercicio 001 Repaso de conceptos elementales

Primera aplicación con AngularJS 
<!-- index.html -->
<html ng-app="app">
  <head>
    <script src="angular.js"></script>
    <script src="script.js"></script>
  </head>
  <body ng-controller="PruebaController">
    <h1>{{mensaje}}</h1>
  </body>
</html>
  • ng-app Esta directiva asocia el DOM con nuestra aplicación de AngularJS, cuyo módulo principal se llama "app". No tiene que estar obligatoriamente en la etiqueta html, puede estar en otra de nivel inferior.
  • angular.js Es el script principal de AngularJS.
  • script.js Contiene el código de la aplicación AngularJS. El contenido se muestra más abajo.
  • ng-controller="PruebaController" Esta directiva asocia la etiqueta que la lleva y su contenido al controlador "PruebaController".
  • {{mensaje}} Las instrucciones entre dobles llaves son evaluadas por Angular y reemplazadas por el resultado de dicha evaluación.
// script.js
var app=angular.module("app",[]);
  
app.controller("PruebaController", ['$scope', function($scope) {
  $scope.mensaje="Hola Mundo";
}]);
  • Método module: sirve para declarar el módulo, la aplicación AngularJS con la que vamos a trabajar. El primer parámetro es el nombre del módulo, que debe coincidir con el que pasamos a la directiva ng-app. El segundo es el array de inyección de dependencias.
  • Método controller: sirve para declarar un controlador dentro del módulo. Recibe como primer parámetro el nombre del módulo, que debe ser el mismo que el que le pasamos a la directiva ng-controller en el marcado con el que se asocia. El segundo parámetro es el array de inyección de dependencias, que incluye la variable $scope que representa el ámbito del controlador y almacena las variables y funciones locales del controlador que pueden enlazarse con el marcado mediante expresiones y directivas.
  • Variable del ámbito de un controlador: Aquí vemos la declaración/asignación de una variable dentro del ámbito ($scope) del controlador.
Uso de expresiones y filtros
Expresión Salida Comentario
2 + 3 = {{ 2 + 3 }} 2 + 3 = {{ 2 + 3 }} Las expresiones entre dobles llaves son evaluadas y reemplazadas por su resultado.
{{ 'AngularJS' | uppercase }} {{ 'AngularJS' | uppercase }} Pueden aplicarse filtros de formato a las expresiones.
{{ 987.654321 | number:2 }} {{ 987.654321 | number:2 }} Ejemplo de filtro numérico
{{ 35.22343 | currency }} {{ 35.22343 | currency }} Algunos filtros aplican la configuración local, que se establece enlazando scripts específicos de la región como "angular-locale_es-es.js"
{{ fecha | date }} {{ fecha | date }} Esta expresión utiliza una variable del ámbito del controlador. Al igual que la anterior depende del script locale.
